# KalaSangham: Reviving India's Forgotten Artforms

[![Live Demo](https://img.shields.io/badge/Demo-Live%20Site-green?style=for-the-badge)](https://kalasangham.vercel.app)
[![GitHub](https://img.shields.io/badge/Repo-GitHub-black?style=for-the-badge&logo=github)](https://github.com/Trisha2910tinaaaaa/KalaSangham)

A digital platform dedicated to preserving and promoting traditional Indian folk artforms like Warli, Madhubani, and Pithora. KalaSangham connects master artists with a global audience, providing them with a sustainable livelihood and ensuring these cultural treasures are not forgotten.

### Prerequisites

- Node.js 18+ installed on your local machine.

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/Trisha2910tinaaaaa/KalaSangham.git
    cd KalaSangham
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) in your browser.
